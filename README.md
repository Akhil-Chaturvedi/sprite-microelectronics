### [Sprite One](Sprite%20One/)

UART-based graphics and AI accelerator for embedded systems. Built on RP2040.

**Status:** v1.1 - Working prototype  
**Use case:** Offload graphics rendering and TinyML inference from resource-constrained MCUs  
**Interface:** UART at 115200 baud
